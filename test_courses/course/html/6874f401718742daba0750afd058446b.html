The staffgradedxblock block type is installed by the staff_graded-xblock repo and presented as Staff Graded Points. Notably and confusingly, this is distinct from MIT's edx_sga block type, which is presented as Staff Graded Assignment.

(It seems like staffgradedblock doesn't provide a template by default, so I haven't gotten a chance to add it here yet. Contributors, please feel free to add one.)
